Configuração do LEGO WeDo 1.0 ou da PicoBoard

Siga esses passos para configurar o Picoboard ou o kit LEGO WeDo 1.0.

(Se você tem a versão mais recente do LEGO WeDo 2.0, veja a Configuração do LEGO WeDo 2.0)

IMPORTANTE: instalar o plug-in

Você vai precisar instalar um plug-in para usar o Scratch com LEGO WeDo 1.0 ou Picoboard. Escolha o plug-in para baixar com base no seu computador e no navegador no qual você usa o Scratch. Se seu navegador não for o Chrome, escolha uma das duas opções a seguir:

  • Windows (outros navegadores)
  • Mac (outros navegadores)

Se você usa o Chrome, você vai precisar baixar e instalar o software em duas partes:

  • Windows (Chrome) e o auxiliar aqui
  • Mac (Chrome) e o auxiliar aqui

Você vai precisar de acesso de administrador em seu computador para instalar o plug-in.

Status: dispositivos de hardware

A luz vermelha significa que o plug-in do navegador não foi instalado.

A luz amarela significa que o dispositivo não está conectado ou não foi detectado.

A luz verde significa que o dispositivo está pronto para uso!

Solução de problemas
Se não estiver funcionando, faça o seguinte:
  • Verifique se você abriu e instalou o arquivo de plug-in correto para seu computador e navegador.
  • Verifique se você configurou a permissão nas preferências do navegador para permitir a execução do plug-in.
  • Reinicie seu navegador.
  • Verifique se você não está com o Scratch aberto em outra guia ou navegador.
  • Verifique se o dispositivo de hardware está conectado.
