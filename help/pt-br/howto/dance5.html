Programar a dança

Diga ao dançarino que passos fazer.

Clique na guia Fantasias.

Cada fantasia tem um passo de dança diferente.

Clique na guia Scripts.

Clique na categoria Aparência e arraste um bloco trocar fantasia para.

Escolha uma fantasia no menu suspenso.

Crie um script de dança trocando fantasias diferentes.

Use um bloco espere (da categoria Controle) para pausar entre os passos.

Clique em qualquer lugar no seu script para testá-lo.

Verifique se você tem um bloco quando clicado no topo do script.
