Define o estilo de rotação do ator

Selecione "esquerda-direita" no menu suspenso para fazer com que o ator gire apenas horizontalmente.


Selecione "ao redor" no menu suspenso para fazer com que o ator vire verticalmente.


Selecione "não girar" no menu suspenso, para que o ator sempre fique de frente para uma única direção.
