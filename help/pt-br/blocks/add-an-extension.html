Adicionar blocos que ampliam as possibilidades do que você pode fazer no Scratch

As extensões permitem que você se conecte a dispositivos físicos ou a serviços web.

In any project, click the More Blocks category and choose Add an Extension.

No futuro, mais extensões serão adicionadas ao Scratch. Para testar nossas extensões experimentais, visite o site scratchx.org

Dispositivos de Hardware

Para usar o LEGO WeDo 2.0, acesse a configuração do LEGO WeDo 2.0.

Para usar o LEGO WeDo 1.0 ou uma PicoBoard, acesse as configurações do LEGO WeDo 1.0 ou da Picoboard.
