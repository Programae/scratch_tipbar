Clique para criar e dar nome a uma nova lista

Quando criar uma lista pela primeira vez, surgirão os blocos de listas. Você pode escolher se a lista é para todos os atores (global) ou apenas para um ator (local).

Você tem a opção de disponibilizar a variável para todos os atores ou apenas para o ator selecionado.

Depois de criar uma lista, oito blocos aparecerão.
